# Web Performance Series | O que é?

Performance para web é uma área do desenvolvimento web focada em criar sites mais rápidos, não apenas sobre o carregamento inicial mas também interações de usuários e carregamentos adicionais

Envolve mensurar e registrar métricas de quão rápido um site carrega, melhorar tempo de carregamento reduzindo latência de respostas HTTP, tempo de acesso a recursos entre o servidor e navegador e performance percebida

Performance para web não é apenas uma lista de itens de melhoria, mas a cultura de criação de sites acessíveis para a maior audiência possível. 
Também deve ser uma preocupação de todos os envolvidos, não apenas dos desenvolvedores. 
Uma boa performance pode ser um recurso do produto, enquanto má performance pode gerar desvantagem e até perda de receita.


### Acessibilidade

A web se torna cada vez mais complexa e maior e com acesso dos mais variados tipos de dispositivos e conectividade. 

Desde os celulares e computadores de última geração a dispositivos antigos.
Por exemplo, um iPhone 8 inicia a renderização de um determinado site em 4,5 segundos e 13 segundos para carregar completamente, já o Moto G4 inicia a renderização do mesmo site em 12,5 segundos e 51 segundos para carregar completamente.

Dispositivos de entrada são mais comuns do que imaginamos. Um site inacessível ou sem interação pode resultar em uma usabilidade frustrante. Se um usuário não se identificar, ou não for paciente o suficiente e algo não funcione como esperado, é provável que ele abandone o site.

Uma pesquisa de 2011 aponta que: 

- 50% dos sites se mantém na mesma velocidade ou mais lento
- 70% dos usuários sente que a navegação é interrompida por sites lentos
- 44% acredita que uma transação lente gera dúvida se realmente funcionou

Uma estimativa mostra que sites que demoram até 10 segundos para carregar podem chegar a 123% de abandono imediato.

A BBC notou que a cada segundo adicional que o site demora, eles perdem 10% da audiência

### Para fechar

De forma geral, web performance é sobre acessibilidade, melhorar conversões, otimização de mecanismos de busca e percepção de performance.
